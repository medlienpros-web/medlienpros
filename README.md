# MedLien Pros (Next.js)

## Run locally
1. Install Node.js (LTS)
2. In this folder:
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Logo
Put your logo at `public/Logo.png` (exact filename) so the app can load `/Logo.png`.

## Deploy (Vercel)
- Push this folder to a GitHub repo
- In Vercel: **Add New Project** → Import the repo → Deploy
